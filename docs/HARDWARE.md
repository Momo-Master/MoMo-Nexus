### Pi-Based Assembly

#### Step 1: Prepare SD Card

```bash
# Download Raspberry Pi OS Lite (64-bit)
# Use Raspberry Pi Imager

# Configure:
# - Hostname: nexus
# - Enable SSH
# - Set WiFi credentials
# - Set username/password
```

#### Step 2: Connect Hardware

```
1. Insert SD card
2. Connect SSD via USB 3.0
3. Connect T-Beam via USB 3.0
4. Connect SIM7600 via USB
5. Attach antennas (BEFORE powering on!)
6. Insert SIM card in SIM7600
7. Connect power supply
```

#### Step 3: Initial Setup

```bash
# SSH into Pi
ssh pi@nexus.local

# Update system
sudo apt update && sudo apt upgrade -y

# Install dependencies
sudo apt install -y python3-pip python3-venv git

# Clone Nexus
git clone https://github.com/Momo-Master/MoMo-Nexus.git
cd MoMo-Nexus

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install requirements
pip install -r requirements.txt
```

#### Step 4: Configure Devices

```bash
# Find device ports
ls -la /dev/ttyUSB* /dev/ttyACM*

# Expected output:
# /dev/ttyUSB0 -> T-Beam (LoRa)
# /dev/ttyUSB1 -> SIM7600

# Test T-Beam
screen /dev/ttyUSB0 115200

# Test SIM7600
screen /dev/ttyUSB1 115200
# Type: AT
# Should respond: OK
```

#### Step 5: Configure Storage

```bash
# Format SSD (if needed)
sudo mkfs.ext4 /dev/sda1

# Mount SSD
sudo mkdir -p /data
sudo mount /dev/sda1 /data

# Add to fstab
echo '/dev/sda1 /data ext4 defaults 0 2' | sudo tee -a /etc/fstab

# Set permissions
sudo chown -R pi:pi /data
```
